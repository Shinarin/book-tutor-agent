# 第三卷 第1章：论财富的自然进步

## 写在前面：我们走到了哪里

如果把《国富论》比作一部交响乐，前两卷是两段宏大的乐章：

- **第一卷**告诉你"财富是怎么生产出来的"——分工、价格、工资、利润、地租；
- **第二卷**告诉你"财富是怎么积累起来的"——资本、货币、银行、节俭。

这两卷给了你一整套**分析工具**。但斯密接下来要做的事情更有野心：他要用这套工具去**审视真实的历史**——为什么有的国家富、有的国家穷？为什么欧洲走到今天这步是这样而不是那样？

第三卷就是这场历史审视。而你现在读到的第 1 章，只是它**简短的开场白**——全章在原书中只有六个自然段，但它树立了一个**理论标杆**，作为后面三章（解释欧洲为何"走偏了"）的对照系。

> **本章核心问题**：如果不受任何人为干预，一个社会的财富发展会按怎样的顺序自然展开？

斯密给出的答案是一个三阶段次序：**农业 → 制造业 → 对外贸易**。这就是他所谓的"财富的自然进步"。

---

## 一、城与乡：分工原理的放大版

斯密开篇说，**任何文明社会最大宗的商业活动，都是城与乡之间的交换**——乡村送出粮食和原料，城镇送回制成品。

这听起来稀松平常，但他接着提醒一件事：**这种交换不是零和的**。

为什么要特别强调这一点？因为在斯密的时代（以及今天的很多人脑子里），都存在一种**朴素的"剥削论"直觉**：城里人不种地、不放羊，吃的喝的全靠乡下，那城里人岂不就是寄生虫？乡下人辛苦劳作的成果，岂不是被城里人白白拿走？

斯密说：**完全反了。** 城与乡是分工原理的一次放大应用：

- 乡下人用自己一份较小的劳动产物（粮食），换回了城里制造的、远超过他们自己制作所需劳动的成品（铁器、布匹、家具）。
- 城里人为乡下的剩余产品提供了一个**市场**——没有城里这个出口，乡下的剩余就只是烂在地里的多余粮食。
- 城越大、市场越广，离城越近的乡村受益越深（省下运费）。

> 🎯 **思考一下**：这个洞察在今天还成立吗？想想你居住的城市与周边的关系。再想想"乡村振兴"政策——到底什么是真正在帮助乡村？是把更多资源砸进乡村，还是把城市做得更繁荣、让乡村离这个市场更近？

### 跨学科联结：分工的"层级递归"

斯密在第一卷讲分工时，讨论的是**车间内部**（制针厂的 18 道工序）。第三卷一开篇，他把同样的逻辑放大到**地理空间**（城与乡）。再往后第四卷，他还会把它放大到**国家之间**（国际贸易）。

这种"同一原理在不同尺度上递归出现"的思想，与现代复杂系统科学中的**分形（fractal）**结构非常相似——同样的规律，在不同层级反复出现。它也呼应了今天经济地理学（Krugman 等人的工作）的核心问题：为什么经济活动会**集聚**？

## 二、生存先于享受：一条几乎像物理定律的次序

斯密的第二个论点非常质朴：

> **吃饭的事，必须先于享乐的事。所以提供食物的产业，必须先于提供便利与奢侈的产业。**

听起来像废话，但它的推论很有力：**城市的存在完全寄生于乡村的"剩余产品"（surplus produce）**。

注意"剩余"这个词——不是乡村的全部产出，而是**养活农民自己之后的那一份**。一个乡村如果只能勉强养活自己，它根本无法支撑任何城市。城市的规模天花板，被乡村的剩余规模所决定。

这条规律在**马尔萨斯**之前就被斯密非常清晰地表达出来了；它后来也成为**农业革命—工业革命因果链**的经典叙事：英国农业的圈地与改良产生了大量剩余食物和被释放的劳动力，才有可能撑起工业城市。

> 🎯 **延伸思考**：在今天的发达国家，农业人口往往只占总人口的 2-5%，却养活了所有人。这是否意味着斯密的论断"城市规模受限于乡村剩余"已经失效了？还是说这条规律只是被**全球化**重新参数化了——今天一个国家的城市可以靠**另一个国家的乡村剩余**来支撑（想想沙特、新加坡、香港）？

## 三、"人的天性"决定了资本投放的优先级

这是本章最核心、也最值得我们带着批判眼光审视的论证。

斯密说，如果让人自由选择，**资本会自发按照这样的优先级流动**：

```
首选：农业（land）
次选：制造业（manufactures）
末选：对外贸易（foreign commerce）
```

### 斯密给出的三条理由

**1. 安全性递减**

- 投资土地：资产看得见摸得着，就在眼前；除非战乱，不会蒸发。
- 投资制造业：资本在工厂里，至少还在自己视线和管理范围内。
- 投资远洋贸易：资本要交给"风、浪，以及更不确定的——遥远国度里你几乎不认识的人的愚蠢与不义"。

**2. 田园生活的天然吸引力**

斯密罕见地流露出一种田园诗的情绪：乡村的美、宁静、独立感"或多或少吸引着每一个人"，因为"耕种土地本就是人类最原初的志业"。

**3. 经验佐证：北美殖民地**

斯密举了一个生动的例子：在北美，土地便宜易得。一个工匠一旦攒下比维持手艺所需更多的本钱，他**不会**用这笔钱去搞规模更大的工厂、做更远的销售，他会去**买一块地，开荒种田**。

斯密解释这个选择的逻辑很有力——

> **工匠是顾客的仆人**，靠给别人提供服务才有饭吃；
> **种植者是自己的主人**，靠自己家庭的劳动养活自己，**独立于全世界**。

这种"独立 vs 依附"的对照，是 18 世纪共和主义政治哲学的核心命题（Jefferson 的自耕农共和理想直接受此影响）。

### 这套论证今天还成立吗？要审慎地批判

斯密这个"自然秩序"是全书最容易被时代抛在身后的论点之一。让我们一条一条来看：

**关于"安全性"**：

在斯密的时代，没有保险、没有股份有限公司（有也不发达）、没有现代金融市场。把资本绑在土地上确实最安全。**但今天**，土地资产并不比股权或债券更安全——城市更新、政策变迁、产业空心化都能让土地资产剧烈贬值；反过来，一个分散化的金融组合往往比一块地更稳健。**斯密的"安全偏好"反映的是前现代的金融环境，而非永恒的人性。**

**关于"田园生活吸引力"**：

这部分今天看更像是个人审美而非经济规律。事实上，**全球城市化**是过去 200 年最强劲的历史趋势之一——人们用脚投票，正好与斯密说的相反，从乡村大规模流向城市。**这不是因为人变了，而是因为城市能提供的机会和收入水平远超农村。**

**关于"农业 → 制造业 → 外贸"的次序**：

这个排序最有问题。**20 世纪后半叶的东亚四小龙（韩国、台湾、香港、新加坡）和日本**，几乎完全是**反着这个次序成功的**——他们恰恰是以**出口导向的制造业**（外贸+制造业）作为起飞引擎，农业反倒是被工业化"拉着走"的。中国 1978 年后的发展路径也是如此。

为什么斯密会错？因为他没有预见到两件事：

1. **全球市场的规模**：当全世界都是你的市场时，"先满足国内需求再出口"的逻辑就失效了。一个小国可以靠出口制造业完成原始积累。
2. **后发优势与技术追赶**：后发国家可以**跳过**前者走过的弯路，直接从落后农业跳到现代制造业（"压缩式发展"）。

但我们也别一棒子打死。**斯密的核心方法论是正确的**——他反对**人为扭曲资本流动的方向**，主张让资本按市场信号自由流动。他错的是把 18 世纪的市场信号当成了永恒的次序。换到今天的市场环境下，斯密自己很可能会承认"工业立国、出口立国"也是一种合理的自然流向。

> 🎯 **思考一下**：如果一个国家明天发现自己资本极度匮乏，斯密的建议（先投农业）和东亚模式的建议（先投出口制造业），到底哪个对？这个问题的答案，可能取决于：(a) 这个国家的人口规模与土地禀赋，(b) 它能不能融入全球市场，(c) 它有没有制度能力运营复杂工业。**没有放之四海而皆准的"自然秩序"，只有具体情境下的"合适秩序"。**

## 四、城市如何自然地从乡村中"长出来"

斯密接下来描绘了一幅田园式的发展图景，非常优美：

农业一旦展开，农民就需要各种**辅助工匠**——铁匠、木匠、车匠、犁匠、瓦匠、皮匠、鞋匠、裁缝。这些工匠彼此之间也需要服务，又不像农民那样必须钉死在自己的田里，**所以他们自然地聚到一起，形成一个小镇**。屠夫、酿酒师、面包师跟着搬来，再加上零售商，小镇就这样长大了。

```
农场需要工匠  →  工匠彼此聚居  →  形成小镇  →  小镇吸引更多工匠和零售商  →  小镇扩张
```

镇上人和乡下人**互为对方的仆人**：小镇是个不间断的集市，乡村人来这里把原产品换成制成品。**工匠能卖多少成品，决定了他们能买多少原料和食物**；而**这种需求又只能随耕作改良的程度而增长**。

注意这里的因果关系：**农业改良 → 农村购买力提升 → 城市制造业扩张**。

如果这一过程不受打扰，城市的规模就会**严格地按比例**跟随农村的改良程度。

### 一个反例佐证：北美殖民地不办大工业

斯密举了一个反例论证：在北美，**任何城镇都没有出现面向远方销售的大制造业**。为什么？因为**土地太便宜**——工匠攒下钱就去开荒了，没人愿意搞大规模制造业。这反过来说明：**大规模制造业是土地稀缺（人口密度高）的产物**。

## 五、欧洲走偏了：自然秩序的"颠倒"

讲完了"自然应该如何"之后，斯密给出本章的**最后一击**：

> **但在所有现代欧洲国家，这种自然秩序在很多方面已经被完全颠倒了（entirely inverted）。**

欧洲的实际历史是：

```
少数城市先做对外贸易  →  引入精细制造业  →  最后才反过来推动农业改良
```

正好是斯密"自然秩序"的反向播放。

为什么会这样？斯密给出一句意味深长的诊断：

> 这是由"原始政府形态所引入的风俗与习惯"所迫——这些风俗在原始政府本身已大改变之后仍然存留。

换句话说：**罗马帝国崩溃后留下的封建制度**，把欧洲农业卡得死死的，资本无法自然流入土地，于是只好绕道流向城市的工商业。

这就是第三卷接下来三章要讲的故事：

- **第 2 章**：为什么罗马崩溃后欧洲的农业被严重抑制？
- **第 3 章**：在农业被抑制的同时，城市为何反而能率先繁荣？
- **第 4 章**：城市的商业又如何反过来改造了乡村？

## 六、本章在全书中的位置

| 卷 | 主题 | 性质 |
|---|---|---|
| 第一卷 | 分工、价格、分配 | 微观—理论 |
| 第二卷 | 资本、货币、银行 | 微观—理论 |
| **第三卷** | **欧洲发展的历史路径** | **宏观—历史** |
| 第四卷 | 重商主义批判 | 宏观—政策批判 |
| 第五卷 | 国家财政 | 宏观—政策建议 |

**第三卷是从"理论"转向"历史"的枢纽**。本章只是一个简短的"标尺"——树立一个"自然秩序应该如何"的对照系，好让接下来三章对欧洲实际历史的考察有据可依。

理解这一点很重要：**斯密并不是在写"欧洲经济史"，他是在用历史来论证一个政策主张**——人为干预（无论是封建残余还是后来的重商主义）会扭曲自然的发展路径，让国家变得不那么富有。

## 延伸思考

1. **"自然秩序"是个时代锚定的概念吗？** 斯密的"自然秩序"实际上是 1776 年时他能观察到的欧洲—北美样本所提示的次序。如果他能看到 20 世纪东亚的崛起，他会修改自己的"自然秩序"理论吗？还是他会说"东亚是非自然路径，只是恰好成功了"？这两种回应背后的哲学差异是什么？

2. **从乡村流出还是流入？** 斯密笔下的人具有"乡村吸引力"——田园美、独立、宁静。但过去两百年人类的实际选择是**大规模城市化**。请尝试调和这个矛盾：是斯密看错了人性，还是他看到的那种"乡村吸引力"在现代社会中以新的形式存在（比如：人们退休后回归田园、富人买乡村庄园、远程办公兴起后的"反城市化"苗头）？

3. **"独立的种植者 vs 依附的工匠"二分今天还成立吗？** 斯密说"工匠是顾客的仆人，种植者是自己的主人"。今天的"自由职业者""独立开发者""创作者经济"是否构成了一种新的"种植者"——既不依附特定雇主，又能跨地理独立工作？这种新形态对斯密的二分意味着什么？

4. **城与乡的关系真的不是零和吗？** 斯密说城乡互利。但今天我们经常听到"虹吸效应"——大城市把周边的人才、资本、年轻人都吸走，导致小城市和乡村空心化。斯密的"互利论"在什么条件下成立，在什么条件下会失效？

5. **方法论层面**：斯密用"如果不受人为干预，事情应该如何"作为批判"事情实际如何"的标尺。这种"反事实历史"的方法论有什么价值，又有什么陷阱？经济学家今天还在用类似方法（比如"自由市场基准"），这种方法是否本身就暗含了某种价值判断？

## 行动指南

这一章本身偏抽象，没有特别明显的"行动"建议。但有一个**思维习惯**值得带走：

> **每当你看到一个发展模式被冠以"自然""必然""规律"的标签时，先问一句：这个"自然"是相对于什么参照系的？换一个参照系，它还自然吗？**

斯密本人是这种批判性思维的大师——他用"自然秩序"批判封建残余和重商主义。但他自己的"自然秩序"，同样需要后人用更新的参照系去重新审视。这正是好的学问应该有的样子：**它启发我们怎么思考，而不是替我们思考。**
