# 第四卷 第6章：论通商条约（Of Treaties of Commerce）

## 引子：这一章在讲什么？

你已经跟着斯密走过了对重商主义的多重审判——奖励金、限制进口、退税、各种贸易特权。这一章是审判的延续，靶子换成了**通商条约（treaties of commerce）**：两个国家之间签订的"互相给点儿好处"的协议。

听起来像是促进自由贸易的好事，对吧？毕竟两国谈判、各让一步、商品互通——多么互利共赢。

但斯密会告诉你：在重商主义时代，这种条约的本质往往是**一方给另一方授予垄断权**，并且——这是最讽刺的——给出垄断权的国家居然把它当成自己的外交胜利来庆祝。本章的核心案例就是 1703 年英葡之间的"梅休因条约"（Methuen Treaty），它被英国人吹捧了近一个世纪，被斯密用一章的篇幅彻底拆穿。

随后斯密还顺手做了一件事——做了一次关于**铸币税（seignorage）**的精彩离题分析。这两部分看似拼接，其实有一条暗线：**都源自重商主义把"金银"误认为"财富"的根本错误。**

---

## 一、通商条约的基本经济学：一份谁吃亏的合同？

设想这样一种条约：A 国答应让 B 国的某种货物免税进口，或以低于其他国家的关税进口；作为交换，B 国答应让 A 国的某种货物进入它的市场。

斯密的分析非常清晰：

### 对"受益国"（被授予特权的一方）

它的商人和制造商在对方市场上享有了**一种垄断地位（a sort of monopoly）**：
- **市场更大了**——别国货物被排除或征更重的关税，所以本国货物销量更多。
- **价格更高了**——少了别国的自由竞争，本国商人可以卖更贵。

### 对"施惠国"（授予特权的一方）

它的商人和制造商被坑了：
- 自己的政府把垄断权"反着授予"了一个外国——**本国生产者在本国市场上要与一个享有特权的外国对手竞争**。
- 本国消费者必须以更高的价格买进口货——因为别国的自由竞争被排除了。
- 本国出口商品换回这些进口货时，单位出口能换到的进口减少了，相当于自己的产品被压价卖出。

### 但施惠国一定吃亏吗？

斯密在这里很谨慎，做了一个微妙的区分。这种"吃亏"通常不是**正损失（positive loss）**，而是**少赚（lessening of the gain）**：

> 虽然施惠国的商品卖得比可能的价格低，但它通常不会卖得低于成本——贸易若真亏本，根本无法长久维持。所以施惠国依然在贸易中获益，只是收益不如自由竞争时多。

📌 **思考题**：为什么斯密在这里特别强调"少赚"而非"亏损"？这跟他在前面章节批评**奖励金（bounties）**时的逻辑有何不同？（提示：奖励金是政府用税款补贴出口，确实会让产品以低于成本的价格卖出去，那才是真正的损失。）

## 二、案例研究：1703 年英葡梅休因条约

### 条约的三条款

斯密原文照录了梅休因条约，简化如下：

| 条款 | 内容 |
|------|------|
| 第一条 | 葡萄牙国王承诺：**永久**允许英国毛纺织品按法律禁止之前的条件进入葡萄牙 |
| 第二条 | 英国女王承诺：**永久**对葡萄牙葡萄酒征收的关税，比同等数量的**法国葡萄酒少三分之一** |
| 第三条 | 程序条款（两个月内交换批准书） |

### 斯密的拆解

读这份条约要看两个关键的不对称：

**葡萄牙的让步：** 只承诺英国毛纺织品"按之前的条件"进入——也就是说，不提高关税。**但没有承诺给英国比法国、荷兰更优惠的待遇。**

**英国的让步：** 葡萄牙葡萄酒的关税比法国葡萄酒少三分之一。这是个**真正的优惠**，因为法国葡萄酒是葡萄牙葡萄酒在英国市场上最直接的竞争对手。

> "由此条约，葡萄牙王室不过被约束承认英国毛纺织品如禁运前的状态——即不抬高税率。但它并未被约束给英国毛纺织品比法国或荷兰更好的待遇。而英国王室则被约束以法国葡萄酒关税的 2/3 进口葡萄牙葡萄酒……所以这条约显然对葡萄牙有利、对英国不利。"

这就是斯密的关键洞见：**条约在文本结构上就是不对等的。** 葡萄牙获得的是**真实的优惠**（比法国低 1/3 的关税），英国获得的只是**回到自由状态的承诺**（不被进一步禁运）。

### 那为什么英国人当时如此自豪？

因为他们用**重商主义的眼光**看：葡萄牙从巴西获得大量黄金，本国消化不了，盈余必然外流。而由于这个条约，**英国成了葡萄牙黄金的主要去处**。据当时的估计，每周从里斯本运抵英国的金子超过 5 万英镑，一年逾 260 万英镑——这数字在 18 世纪是天文数字。

英国人觉得：**我们用毛纺织品换回大量黄金，岂不是赚翻了？**

## 三、斯密的根本反击：黄金是商品，不是财富本身

这就是本章最有力的一击。斯密说，**就算把所有这些黄金都算到英国头上（事实上其中很大部分要转运给其他欧洲国家），这个贸易也并不比其他任何贸易更有利。**

为什么？因为：

### 第一，黄金大部分要再出口

英国每年留作铸币和金器（plate）的黄金量很小，绝大部分黄金会**再被运出去换其他国家的消费品**。这是一种典型的**迂回贸易（round-about trade）**：

```
英国货物 → 葡萄牙黄金 → 其他国家的消费品 → 英国
```

而如果英国直接生产出适合那些"其他国家"市场的货物，省去中间这一步：

```
英国货物 → 其他国家的消费品 → 英国
```

**直接贸易（direct trade）所需的资本远少于迂回贸易**。少占用的资本就可以投到别处去激发更多产业、生产更多年产物。

📌 这里斯密在重申他在第四卷前几章反复阐述的一个原则：**贸易的目的是为了获得消费品，而非为了积累金银**。判断一笔贸易是否有利，不看它带回多少金银，而看它最终带回多少能消费、能使用的实物，以及它占用了多少资本。

### 第二，就算切断葡萄牙贸易，黄金还是会到英国手里

葡萄牙国内消化不了那么多黄金，盈余**必然外流**。如果英国买不到，别国会买，再以正常价格卖给英国。

> "黄金，如同任何其他商品，总是能在某处被那些愿意付出等价物的人买到。"

英国从葡萄牙买黄金是"第一手"，从别国买是"第二手"，可能稍贵——但这点差异**根本不值得国家政策为之挂怀**。

### 第三，"贸易差额"这个观念本身就是糊涂账

斯密在这里给出一个漂亮的论证：

> 一国对黄金的**有效需求**是一定的。如果 9/10 的黄金来自一国，那么剩下 1/10 才能来自所有其他国家。从某些国家进口的黄金越多，从其他国家进口的就越少；同时**对那些国家的"贸易差额"看起来就越不利**。

换言之，重商主义者只盯着对葡萄牙的贸易差额顺差欢欣鼓舞，却忽略了：这同一笔金子流入，机械地让英国对其他国家的"贸易差额"显得更糟。**这只是会计上的把戏，不是真实的利益变化。**

> "现代政策中最无意义的对象——贸易差额（the balance of trade）——在一国对某些国家显得顺差越大，必然在另一些国家显得逆差越大。"

📌 **跨学科联系**：这正是现代经济学中"**多边贸易差额无意义**"原则的最早表述。一国对某国的双边逆差，只是它在世界生产链中所处位置的结果，与该国整体经济健康基本无关。今天每当政客拿着"我们对某国贸易逆差 X 千亿"做文章时，斯密的批判依然有效——他在 1776 年就讲得很清楚了。

### 第四，金银确实有用——但只是作为"贸易工具"

斯密承认葡萄牙贸易**确实有些好处**，但这些好处不是它带回大量黄金本身，而是黄金这种商品在迂回贸易中**特别方便**：

- 任何商品都能用金银换得，因为它们是"通用的商业工具"；
- 体积小、价值大，运输成本低；
- 长途运输价值损耗少。

所以英国把葡萄牙黄金当作**润滑全球贸易的工具**，这是个**值得肯定但绝不重大**的优点（"considerable but not capital"）。

### 第五，最辛辣的政治讽刺

斯密在这里讲了一个故事。七年战争末期，法国和西班牙要求葡萄牙国王驱逐英国船只、接纳法西驻军。如果葡萄牙国王答应——英国人当时觉得是大灾难，因为"英国失去葡萄牙贸易就活不下去"。

斯密冷冷地说：

> 如果葡萄牙国王屈服于那些屈辱条件，**英国反而摆脱了一个比失去葡萄牙贸易大得多的负担——即支援一个虚弱盟友的负担**。这个盟友自身防御如此空虚，以致英国倾全力恐怕都难为他多撑一个战役。葡萄牙贸易的丧失，至多让当时从事此贸易的商人在一两年内找不到同样有利的资本去处——也就这点不便。

这是经典的斯密笔法：**用经济理性戳破政治幻象。** 重商主义者把"葡萄牙贸易不可失去"的迷信，捧成英国必须为之承担巨额军事和外交成本的战略目标，而本质不过是"几位商人需要一两年时间转移资本"。

## 四、批判性审视：斯密说对了什么？说漏了什么？

### 仍然成立的核心洞见

1. **双边贸易差额本身没有意义**——这一点 250 年来从未被推翻，只是反复被政客重新发明、又被经济学家反复批驳。
2. **直接贸易优于迂回贸易**（在贸易成本意义上）——这是后来贸易理论中"贸易成本"概念的雏形。
3. **垄断与排他特权的本质是把"少数人的利益"包装成"国家利益"**——这是斯密贯穿全书的政治经济学批判。

### 需要补充或修正的地方

1. **斯密低估了"互惠条约"作为打破贸易壁垒的工具的潜力**。他批评的梅休因条约确实是个不平等的垄断协议，但现代意义上的**自由贸易协定**（如 WTO 体系）通过**最惠国待遇**条款，恰恰是用条约工具实现普遍降税——这是斯密时代尚未出现的制度创新。

2. **斯密把"获得葡萄牙葡萄酒便宜"当作纯粹的损失**，但从消费者剩余角度看，英国葡萄酒消费者**确实因为关税优惠而获益**——只是这部分获益没有进入斯密的天平。现代福利经济学会更细致地核算。

3. **国家间的安全联盟有其独立价值**。斯密用纯经济学嘲讽英国维持葡萄牙的"负担"，但 18 世纪英国的全球战略地位与葡萄牙在大西洋的港口确实息息相关——这超出了纯贸易计算。

不过总的来说，**斯密的核心论证是稳固的**：判断一份通商条约的得失，不能用"换回了多少金银"，而要用"动用了多少资本、换回了多少可消费的实物、是否限制了竞争"。这是经济学思维方式的根本胜利。

## 五、离题：铸币、铸币税与"佩涅罗珀的织布"

本章后半段是个看似无关的离题——斯密谈起了铸币和**铸币税（seignorage）**。为什么放在这一章？因为：

> "鼓励铸币的法律源于重商主义引入的庸俗偏见。它就像是一项**对货币生产的奖励金（bounty upon the production of money）**——而那正是重商主义心目中所谓的'财富'。"

也就是说，在斯密看来，**英国政府承担铸币费用**（让私人把金银带去铸币厂免费铸成金币）这件事，本质上跟"给出口黄金商发奖励金"是一回事——都是重商主义把"金银 = 财富"误解的副产品。所以这个话题归在重商主义批判之下。

### 核心概念：什么是铸币税？

- **铸币税（seignorage）**：政府对铸币收取的费用，即"金条→金币"过程中的加工费。
- 如果**没有铸币税**（英国情况）：1 盎司金条的市场价 ≈ 1 盎司金币（只差去铸币厂跑一趟的麻烦和等几周时间）。
- 如果**有铸币税**（法国情况，8%）：1 盎司金币比 1 盎司金条贵 8%，因为加工费抬高了它的价值。

### 问题：为什么没有铸币税会出大麻烦？

斯密提供了一个精彩的分析。当时英国流通的金币普遍**磨损不足重**（短缺约 2%），但因为按"枚数计算"（by tale）而非按重量交易，所以新铸的足重金币和磨损过的旧金币在市场上**面值一样**。这就出现了套利机会：

```
1. 把一磅重的足重金币（44.5 几尼）"凭面值"使用 → 价值 £46:14:6
2. 把同样的金币熔掉变成一磅金条 → 在市场上能卖 £47:14:0 到 £48
3. 用 £46:14:6 的金币熔出 £47:14:0 的金条，无风险套利 1 镑！
```

结果是：**铸币厂白天造出的新币，晚上就被熔掉了**。

斯密给了一个非常文学化的比喻：

> "铸币厂的运作就像**佩涅罗珀的织布（the web of Penelope）**——白天织好，夜里又被拆掉。"

📌 **跨学科联系**：佩涅罗珀是荷马史诗《奥德赛》中奥德修斯的妻子，她为了拖延向求婚者答复，谎称要织完一件寿衣才出嫁，白天织晚上拆。斯密用这个典故是绝妙的——铸币厂年年忙碌、产量惊人（£85 万/年），实际上只是在补回头一晚被熔掉的部分，并没有真正增加流通货币。

### 解决方案：合理的铸币税

斯密论证：**适度的铸币税能消除这种套利**。

逻辑链：
1. 假设铸币税 5%——那么金币就比同重金条值钱 5%。
2. 即使金币磨损了 2%——它依然比金条值钱 3%（5% - 2%）。
3. **熔铸新币变金条会亏 3%**——套利消失。
4. 同理可推：铸币税 = 磨损率时，套利收益为零；铸币税 < 磨损率时，套利缩水但未必消失。

### 谁付铸币税？

斯密给出一个常被忽略的洞察：

> **没人最终付铸币税**——因为对货币而言，每个人既是买家也是卖家（"every man is a merchant"），没有最终消费者。每个人把铸币税"垫付"进货币的市场价值里，下次花出去时又收回来。

所以**铸币税本质上是无成本的财政手段**——政府收得到钱，民众实际上不损失。当时英国政府之所以坚持免铸币税，主要是为了讨好**英格兰银行**（铸币的最大客户）——但斯密指出，连英格兰银行也想错了自己的利益：如果未来金币又磨损了，没有铸币税意味着银行每年要承担巨额损失（估计达 £21,250 一年）。

### 批判性评估

斯密的铸币税分析在 18 世纪是**第一流的货币经济学**。它正确地区分了：
- **货币的实际含金量（real value）**
- **货币的名义价值（nominal value）**
- **铸币税对二者关系的调节作用**

后世的货币理论（尤其是 19 世纪的金本位讨论）大量延续了斯密的框架。今天纸币时代当然不再有"熔币套利"的问题，但**铸币税（seignorage）这个词被保留下来**，指中央银行通过发行货币（生产成本几乎为零、面值远高于成本）获得的收益。**现代央行的铸币税正是当年斯密分析的逻辑后裔**——只不过铸币的"加工费"如今几乎等于面值。

## 六、本章总结：一个统一的批判

把通商条约与铸币税放在同一章，看似拼接，实则斯密在做一件事：**揭示重商主义"金银 = 财富"这一根本错误的两种表现**。

| 表现一：通商条约 | 表现二：免铸币税 |
|----|----|
| 为了多换回金银 | 为了多造出金币 |
| 给外国授予垄断权，自己吃亏 | 给私人客户（主要是银行）一种隐性补贴 |
| 误以为顺差就是国家变富 | 误以为铸币越多就是国家变富 |
| 反复签订不平等条约 | 铸币厂"佩涅罗珀式"无效工作 |

斯密的统一回答是：**财富是劳动产品和年产物，不是金银存量**。一旦扎根这个观点，无论是通商条约还是铸币政策，都会被重新审视、得出更合理的结论。

## 延伸思考

1. **斯密对"双边贸易差额"的批判，今天还成立吗？** 当代的"中美贸易战"、"美欧贸易摩擦"几乎都聚焦在双边逆差。如果你给当今的贸易决策者写一份斯密视角的备忘录，你会怎么说？而经济学家们通常给出的回答（"双边逆差不重要，多边逆差才重要"）和斯密的回答一样吗？

2. **梅休因条约真的对英国不利吗？** 历史学家有不同看法。有人指出，葡萄牙葡萄酒进入英国 → 英国发展出对葡萄牙红酒（如波尔图酒）的偏好 → 形成了波尔图酒贸易和品牌资产 → 这些动态效应可能远超斯密的静态计算。当你判断一项贸易政策是好是坏时，应该用静态分析还是动态分析？两者冲突时怎么办？

3. **铸币税的现代版本**。今天美元作为世界储备货币，美联储发行美元、世界各国接受、本质上是"全世界都在用劳动产品换美国印钞机生产的纸"。这种"国际铸币税"是否就是斯密所批判的"用劳动换金银"的现代翻版？或者它已经发生了根本性质的变化？

4. **"任何来自商人的政策建议都要怀疑地审查。"** 这是斯密在第一卷第 11 章末的著名告诫。本章揭示的"商人把自己的利益包装成国家利益"——梅休因条约下从事葡萄牙贸易的商人、推动免铸币税的英格兰银行——是这条原则的鲜活注脚。在你熟悉的领域（科技、金融、医药、能源……），哪些"国家利益"叙事其实是某个利益集团的包装？

5. **"佩涅罗珀的织布"是一种深刻的隐喻**。除了铸币之外，社会中还有哪些活动是这种"白天织、晚上拆"的循环？（教育系统的某些环节？医疗体系？官僚机构的某些工作？）能找到这种隐喻的，往往意味着制度设计存在套利空间。

## 行动指南

- **读国际贸易新闻时**：每当看到"我国对 X 国贸易顺差/逆差扩大"时，问自己：这背后是结构性问题，还是会计幻觉？同时对其他国家的差额是否相应缩小/扩大？整体多边差额是否真的恶化？
- **判断"互利"协议时**：仔细看条款的对称性。对方做出的是**真实让步**（如降低关税）还是**虚假让步**（如承诺不进一步加税）？双方各自获得的是真正的开放还是排他性的特殊待遇？
- **理解中央银行政策时**：现代货币体系中的"铸币税"概念依然有效。当一国央行通过印钞融资政府开支时，本质上是斯密铸币税分析的现代延伸——理解谁最终承担成本（持有该货币的国内外人民）。
- **警惕"国家利益"包装**：每当某个政策被论证为"必须如此否则国家会蒙受巨大损失"时，先识别出谁是直接受益者，再判断"国家利益"叙事的可信度。这是斯密给后世最持久的政治经济学方法论。
