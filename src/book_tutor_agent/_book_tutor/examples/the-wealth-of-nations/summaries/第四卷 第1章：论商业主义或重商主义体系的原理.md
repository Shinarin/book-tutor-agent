## 第四卷 第1章：论商业主义或重商主义体系的原理

本章是斯密对"重商主义"（mercantile system）的系统性批判的开篇。他先剖析"财富即金银"这一流行观念的来源与谬误，再论证一国并不需要政府刻意囤积贵金属，最后引出重商主义为追求顺差所采用的六大政策工具，作为后续各章逐一驳斥的纲领。

### 一、"财富即金银"观念的起源

- **货币的双重职能造成误解**：货币既是流通工具，又是价值尺度。日常语言中"富有"几乎等同于"多金"，于是"国家变富 = 积累金银"的观念自然形成。
- **历史例证**：西班牙人到新大陆只问"有无金银"；鞑靼人则问"有无牛羊"——在游牧民族眼中，牛羊才是财富。斯密尖刻地指出，**鞑靼人的看法或许更接近真理**。
- **洛克（Locke）的论证**：金银不像其他易耗商品，是"稳定的朋友"，故应将增殖金银定为政治经济学的首要目标。
- **战争论据**：另一派承认对内贸易中金银多少无关紧要，但坚称对外战争与维持远方舰队军队必须以金银支付，因此和平时期就要积累。

### 二、欧洲各国的限制金银出口政策与商人的反驳

- 西班牙、葡萄牙严禁金银出口；法、英、苏格兰也曾有同类法律。
- 当商业兴起，商人发现禁令妨碍贸易，遂提出两点反驳：
  1. **托马斯·孟（Mr Mun）的"农耕比喻"**：出口金银就像农夫播种，最终从国外换回更多财宝。其著作《英国得自对外贸易的财富》（*England's Treasure in Foreign Trade*）成为政治经济学的基本信条。
  2. **贸易差额（balance of trade）理论**：金银小巧易走私，禁令无效；唯一手段是出口大于进口，从而由外国以金银偿付差额。
- 斯密评判：商人的论证"部分正确，部分诡辩"——正确之处在于承认金银出口往往有利、且禁令无效；诡辩之处在于把保有金银视为政府须特别关注的事项，而对其他有用商品却不必如此。

### 三、政府无需干预金银供给

- **核心原则**：一国买金银和买葡萄酒一样，只要有购买力，自由贸易自会按"有效需求"（effectual demand）供应所需数量。
- **金银流动最为容易**：体积小、价值大，最易被搬运到价格高的地方；西、葡的严刑峻法挡不住金银外流，斯巴达的禁律也阻止不了金银流入。
- **货币短缺多为伪问题**：人们抱怨"缺钱"，实则缺的是信用与可交换之物。**过度交易**（over-trading）才是商业城镇"缺钱"叫嚷的真正成因。
- **替代手段充裕**：若金银真的不足，可用以物易物、商业信用相互冲抵、以及"管理良好的纸币"加以补救。
- **结论**：**财富不在金银，而在金银能买到的东西**。货币只是国民资本中最小且最不获利的部分。

### 四、贵金属是工具而非财富本身

斯密用"锅碗瓢盆"作比：

- 一国的金银数量自然受其"用途"（充当货币流通、作为器皿装饰）决定。
- 试图通过非常手段堆积超额金银，等同于强迫家庭多置无用厨具——只会减少养活和雇用人民的真正财富。
- 多余的金银因其易于运输，**任何法律都拦不住它流出国门**。

### 五、对外战争并不靠囤积金银支付

- 舰队和军队是用"可消费物资"维持的，而非黄金白银。
- 一国可以三种方式向远方军队付款：
  1. 输出已积累的金银（包括流通货币、私人器皿、君主财库）；
  2. 输出本国制造业产品；
  3. 输出本国粗产品（rude produce）。
- 三种"积累的金银"在实践中作用都很有限：
  - 流通货币不能多撤，撤出后由纸币（exchequer notes、navy bills、bank bills）补位；
  - 私人器皿熔铸所得微不足道；
  - 除普鲁士国王外，欧洲君主已不再积累财库。
- **真实案例**：上次法国战争英国耗费 9000 万英镑以上，超过 2/3 花在海外。若全靠本国流通金银（最多估 3000 万）周转，则该金银必须在六七年间外流又回流两次以上——足以证明政府监管货币之多余。
- **结论**：战争费用主要靠输出英国商品（尤其是精细制造品——"体积小、价值大"）来支付；粗产品因运费高、本国又需自食，几乎无法用作远征支付。
- **"商业共和国之货币"**（money of the great mercantile republic）：在各大商业国之间循环的金银块（bullion），是国际贸易的支付工具，但归根结底仍由各国土地与劳动的年产物购得。

### 六、对外贸易的真正利益

对外贸易的好处**远不止**输入金银。任何两国进行贸易都获得两项利益：

1. **为本国过剩产品找到出路**，并换回本国需要之物；
2. **突破国内市场狭小的限制**，使分工得以发挥到极致，从而提高劳动生产力、增加年产物。

发现美洲和绕过好望角通往东印度，对欧洲的真正贡献：
- **不是**带来便宜的金银本身（金银变便宜只是微小便利）；
- **而是**开辟广阔新市场，促成新的分工和技艺进步，提升了欧洲的真实收入与财富。
- 东印度贸易因被各国特许公司垄断（葡—荷—英—法—瑞—丹相继效仿），益处远不如对美贸易。

### 七、重商主义体系的两条基本前提与六大政策工具

经过反复批判，斯密总结重商主义建立在两条前提上：

1. 财富在于金银；
2. 无矿的国家只能靠"贸易顺差"（出口大于进口）获得金银。

由此衍生出政治经济学的两大目标：**最大限度减少进口，最大限度扩大出口**。其六大政策工具是：

| 类别 | 工具 |
|---|---|
| **进口限制**（两种） | ① 限制本国可生产的同类外国商品的进口 |
| | ② 对贸易差额"不利"国家的几乎所有商品予以限制 |
| **出口鼓励**（四种） | ③ 出口退税（drawbacks） |
| | ④ 出口奖励金（bounties） |
| | ⑤ 有利的对外通商条约 |
| | ⑥ 在远方建立殖民地（往往附带垄断权） |

斯密宣告：以下各章将逐一考察这六种手段，**不再纠缠它们是否带来金银，而专注于评估它们对一国年产物的实际影响**——因为正是这一年产物的增减决定了国家真正的财富与收入。
